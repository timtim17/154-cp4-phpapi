CP4 Message Board

API Doc

GET /messageboard.php

Response Type: JSON

Responds with an array of every message on the message board. If there are no messages, an empty array is sent back.

Required Parameters: None

Example Request

GET /messageboard.php
[
    "Mowgli <3 Debug Duck",
    "CSE 154 is awesome!"
]

POST /messageboard.php

Response Type: JSON

Takes a message via POST and adds it to the message board. Parameters must be sent within the body of the request as form-data. Requires authentication: a valid API key/password must be provided within the auth parameter.

Required Parameters: None

Example Request

POST /messageboard.php
[
    "Mowgli <3 Debug Duck",
    "CSE 154 is awesome!"
]
